Genshin's Sky Limit

This program aims to convert JSON files created by Sky Studio as well as music sheets in Sky's ABC1-5 notation into a proposed ABC1-7 notation for Genshin Impact to use for Windsong Lyre. Gameplay for the new gadget can be found here.

If the program does not work, please make sure that there are no brackets [] at the start and end of the text!
